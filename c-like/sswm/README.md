# sswm

## function

- `MOD4 + d` to open dmenu
- `MOD4 + Shift + q` to kill a window
- `MOD4 + Shift + d` to close sswm
- `MOD4 + Button1` focus window and move it
- `MOD4 + Button3` resize window
